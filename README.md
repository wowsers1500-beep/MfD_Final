# MSE Interactive Web App

An interactive web application for learning about Mean Squared Error (MSE) through 4 interactive graphing questions.

## Features

- **Interactive Graphs**: Visualize data points and fitted models (horizontal lines, linear functions, and parabolas)
- **Editable Points**: Modify x and y coordinates of points using text input controls
- **Live MSE Calculation**: See MSE values update in real-time as you adjust points
- **Worked Solutions**: Click "Reveal Answer" to see detailed step-by-step solutions
- **4 Questions**: 
  - Q1: Horizontal line with 3 points
  - Q2: Linear function with 3 points
  - Q3: Linear function with 4 points
  - Q4: Parabola with 4 points

## How to Use

1. Select a question from the dropdown menu
2. Edit the x and y coordinates of any point using the input controls below the graph
3. Watch the graph update as you make changes
4. Click "Reveal Answer" to see the complete MSE calculation with a detailed breakdown
5. Click "Hide Answer" to hide the solution and try again

## Technical Details

- **No frameworks**: Built with vanilla JavaScript, HTML, and CSS
- **Front-end only**: No server or backend required
- **Responsive design**: Works on desktop and tablet browsers
- **Real-time updates**: MSE calculations update instantly as you modify coordinates
